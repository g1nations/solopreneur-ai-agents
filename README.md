## 🎯 소개 (Introduction)

**Solopreneur AI Agents**는 1인 기업가(Solopreneur)와 인디 개발자를 위해 설계된 **AI 페르소나 컬렉션**입니다.

딱딱한 AI 대신, 개성 넘치는 AI 동료들과 함께 일하며 업무 효율과 재미를 동시에 잡으세요!

### ✨ 특징

- 🎭 **생동감 있는 캐릭터**: 상황에 따라 다양한 표정으로 반응
- 💼 **1인 기업 특화**: 빠르고 실용적인 해결책 제시
- 🔧 **쉬운 설치**: `.cursorrules` 파일 하나로 즉시 사용

## 🛠️ 설치 방법 (Installation)

### Cursor 사용자 (추천)

1. 원하는 에이전트 폴더로 이동
2. `.cursorrules` 파일을 복사
3. 당신의 프로젝트 루트에 붙여넣기
4. 끝! AI에게 말을 걸면 해당 에이전트가 응답합니다.

### ChatGPT / Claude 사용자

'Custom Instructions' 또는 'System Prompt'에 `.cursorrules` 내용을 붙여넣으세요.
